# LIDO

[Lightweight Information Describing Objects](https://lido-schema.org/) is an XML harvesting schema.

Be wary of any system that has the word "lightweight" in its name or description. It seldom is. Be that as it may, LIDO has been used as a standard in many cultural heritage contexts (mostly museums).

As it is so broadly used in the German cultural heritage sector, we have to provide robust support for importing LIDO datasets.

There is a current set of working code in the [Hydra scraper](../hydra.md), that has been used to import data from several datafeeds. _needs links_

There is also [this Github repo](https://github.com/epoz/lidolator), that has tried to gather useful info and provide some analysis utilities.

On [this link](/tech/lido_xpaths_DDB.html) is an analysis of all the LIDO XML source files in the Deutsche Digitale Bibliothek (as of April 2026)
