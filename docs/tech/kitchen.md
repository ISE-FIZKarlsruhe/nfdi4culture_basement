# The Kitchen

The ETL pipeline for NFDI4C is managed by [the Kitchen](https://gitlab.rlp.net/adwmainz/nfdi4culture/knowledge-graph/culture-kg-kitchen)

The Knowledge Graph Kitchen is a versatile ETL (Extract, Transform, Load) environment designed for efficient consuming, processing, integration, and analysis of data into a Knowledge Graph. It comprises several Python components, each serving a unique purpose within the "kitchen" to consume, clean, version control, publish and analyze data in the KG.
